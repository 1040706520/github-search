## github search helper是什么
是github高级搜索的简洁版,可以方便的定义搜索条件

## 如何使用
* 克隆
* chrome://extensions
* 打开开发者模式
* 点击加载已解压的扩展程序 -> 选择上面克隆的地址指向dist目录(如：xxx\github-search\dist)
